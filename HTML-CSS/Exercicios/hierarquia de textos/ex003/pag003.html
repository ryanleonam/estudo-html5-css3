OL / UL

Listas Ordenadas

Segundo o material de estudos do Guanabara:

A HTML chama de ordered lists todas aquelas listas onde a ordem dos itens é algo muito importante. Um passo-a-passo para criar um bolo, uma lista de aprovados no vestibular e uma lista com os carros mais caros do mundo são exemplos de listas ordenadas.

Para criar uma ordered list ultilize a tag <ol> e a tag <li> para identificar os itens da lista

Exemplo:

    Tarefas do dia
  1. Acordar
  2. Café da manhã
  3. Escovar os dentes
  4. Lançar notas
  5. Levar filho à escola

Você pode usar o parâmetro TYPE para determinar os marcadores da lista, como A para listas alfabéeticas, 1 para numeradas e I para algarismos romanos.

Usando o parâmetro START você da inicio a contagem da lista

Listas não Ordenadas

Para listas onde a ordem não importa ultilize a tag <ul>

Exemplo:

    Lista de compras
  • Banana
  • Maçã
  • Achocolatado
  • Carne
  • Sabão

Tipos de marcadores para as UL são "circle", "disc" e "square"

Listas Mistas e de Definição

Crie UL dentro de OL ou vice-versa.

Exemplo:

    Restaurantes
  1. Cosi
    • Croquete de pato
    • Rotti
    • Massas
  2. SouthSide
    • Camarão empanado
    • Risini de camarão
    • Gyosa de rabada
  3. Street Food
    • Cachorro quente
    • Hamburguer
    • Pastel

Lista de definições

Ultilize a tag <dt> para o termo e <dd> para a definição

Exemplo:

Termo
  1. Unidade do léxico; vocábulo, palavra.
  2. Fim, remate ou conclusão no espaço ou no tempo.
Definição
  1. Ato ou efeito de definir(-se)
  2. Delimitação exata, estabelecimento de limites.
