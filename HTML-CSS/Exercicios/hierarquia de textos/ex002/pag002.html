Entendendo hierarquia de titulos

Principais formatações

Negrito / Destaque

Para colocar um termo em negrito ultilize a tag <b>(não semantica).

Para um termo semantico em destaque, ultilize a tag <strong>. Que torna um termo forte.

Itálico / Ênfase

Para ultilizar um termo em itálico use a tag <i>.(Não semantica).

Para um termo em ênfase com uma tag semântica ultilize <em>

Texto marcado

Para marcações ultilize a TAG <mark>, mas para estilizar a cor de suas marcações ultilize a TAG <style=background-color> no head.

Tamanho de texto

Para letras miúdas ultilize a TAG <small>.

Texto excluso e incluso

Para tornar um texto excluso, ou seja, deixá-lo legivel porem riscado use a TAG <del>. Para mostrar que um texto foi inserido depois use a TAG <ins>

Textos sobrescritos e subescritos

Para textos sobrescritos use a TAG <sup> e <sub¨> para sobescritos

Outras formatações

Código-fonte / Pré-formatação

Para tornar o texto mono-espaçado ultilize a tag <code>

Para exibir um código com identação ultilize a tag <pre>

Citações simples / Citações completas

A tag <Q> é semantica e de melhor uso que as aspas em citações pois da sentido.

Use a tag <blockquote> para citações em bloco. E dentro da tag é interessante usar o parametro "cite" dentro da tag para linkar o material de citação melhorando os mecanismos internos do seu site

Abreviações

Para esclarecer abreviaçõees como "HTML" ultilize a tag <abbr>
