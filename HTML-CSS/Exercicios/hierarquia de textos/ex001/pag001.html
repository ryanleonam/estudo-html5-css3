Para usar imagens da internet no seu próprio site é preciso ficar atento aos direitos autorais

É necessario também ficar muito atento ao tamanho dos arquivos usados no site, um site pesado tem menos chances de aparecer no google por conta do trafego de banda
Por isso é necessario redimensionar as imagens em algum editor para deixar o site mais leve

Para adicionar uma imagem ao projeto, crie uma pasta dentro do projeto com a imagem que deseja inserir de use a tag <img>

para carregar imagens externas copie o endereço da imagem e adicione ao source da tag <img>

Agora adicionaremos o FAVICON que é o icone apresentado na URL.
No HEAD adicione <link favicon> e no href busque a imagem que deseja com ctrl+espaço
