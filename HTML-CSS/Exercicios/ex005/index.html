Links externos

Links externos nos levam para outros sites usando a tag <a>

Quando for abrir sites externos é interessante que abra-os sempre em uma nova aba, usando os parâmetros target="_blank" e rel="external" ou rel="nofollow" para links não apontados por sites de pesquisa.

Links internos

Para links internos utilize o parâmetro rel="next" para a proxima pagina e rel="prev" para a pagina anterior.

Para acessar ao primeiro exercicio do curso clique no link, lá encontrará metódos de inserção de imagens, icones e outras informações
